## Trabajo Práctico N°2 – Teoría de Algoritmos

Este trabajo práctico tiene como objetivo aplicar diferentes paradigmas de diseño algorítmico a través del desarrollo de cuatro ejercicios. En cada uno se abordaron distintas estrategias de resolución: Programación Lineal, Redes de Flujo, Algoritmos de aproximación y Algoritmos randomizados implementando algoritmos específicos que cumplen con las consignas planteadas por la cátedra.

Para cada ejercicio se incluye el código fuente correspondiente y los resultados obtenidos.
